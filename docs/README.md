# Documentation

Training materials, design briefs, and reference documents for the TACTRACE Trainer.

## Planned Contents

- **`TACTRACE_V1.4_Training_Guide.pdf`** — Complete instructor + trainee guide updated for V1.4 (4 test points, breaker, V1.4 fault tables).
- **`Trainee_Worksheet.pdf`** — Standalone trainee worksheet for an unknown-fault diagnostic exercise.
- **`Instructor_Reference.pdf`** — Instructor-side reference: setup procedure, fault-mode tables, evaluation rubric.
- **`Quickstart.md`** — One-page quickstart for a maintainer who has never seen the device.
- **`V2_design_brief.md`** — Formal V2 design brief: requirements, architecture options, open questions, risk register.

## Status

A complete V1 13-page training guide has been authored and is being revised to V1.4 before publication here. Documentation will be added as it is updated and reviewed.

## Conventions

- Published documents are PDF or Markdown. Source files (Pages, Word) are kept locally; only exported PDFs are committed.
- Documentation revisions are tracked in [`../CHANGELOG.md`](../CHANGELOG.md) when they are released alongside a hardware revision.
- Claims that have not yet been validated by measured testing or peer review are marked **"Needs peer review"**.
